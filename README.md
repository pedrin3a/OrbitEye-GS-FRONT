#  OrbitEye Landing Page

##  Sobre o Projeto

OrbitEye é uma Landing Page desenvolvida em HTML e CSS inspirada em tecnologias espaciais e sistemas modernos de monitoramento climático.

O projeto apresenta uma solução voltada para monitoramento ambiental em tempo real, utilizando conceitos relacionados à indústria espacial, satélites e análise de dados climáticos.

---

#  Objetivo

Demonstrar como tecnologias espaciais podem auxiliar na prevenção de desastres ambientais por meio do monitoramento climático e emissão de alertas.

---
